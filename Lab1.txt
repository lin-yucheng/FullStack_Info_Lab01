1. Change Directory: cd <dir>
2. Create a Directory: mkdir <dir>
3. Rename a Directory: mv <old_name> <new_name>
4. List the contents of a directory: ls
5. Move a file to a new location: mv <file_name> <location>
6. Copy a file to a new location: cp <file_name> <location>
7. Search for a string in a file: grep -r "<string>"
8. Access the shell of a remote machine: ssh root@<IP address>
9. Remove a directory folder: rm -rf <dir>
